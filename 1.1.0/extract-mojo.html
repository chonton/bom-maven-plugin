bom:extract

Full name:

org.honton.chas:bom-maven-plugin:1.1.0:extract

Description:

Create a Bill of Materials from a set of jars or tar file.

Attributes:

  • The goal is not marked as thread-safe and thus does not support parallel builds.

Optional Parameters

Name Type Since Description
<artifactId> String - The artifactId for the bom.
Default value is: bom.
User property is: bom.artifactId.
<bom> String - The name of the Bill of Materials file
Default value is: bom.xml.
User property is: bom.bom.
<groupId> String - The groupId for the bom.
Default value is: extracted.
User property is: bom.groupId.
<source> String - The source directory or tar.
User property is: bom.source.
<unknowns> String 1.1.0 The directory to receive a copy of unknown jars
User property is: bom.unknowns.
<version> String - The version for the bom.
Default value is: 1.0.0-SNAPSHOT.
User property is: bom.version.
